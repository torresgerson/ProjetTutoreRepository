1. Choix du développement

  • Tout d’abord, il faut choisir les technologies à utiliser pour la réalisation du projet, ils doivent comprendre en tant que le développement mobile, la base de données et le capteur d’alerte en cas d’un antivol du vélo.

    Développement mobile

    Onsen UI: Onsen UI est un Framework open source pour créer des applications Web progressives (PWAs) et des applications hybrides. Il est basé sur PhoneGap et Cordova.

    Il peut être utilisé avec plusieurs frameworks en tant de JavaScript, AngularJS (1, 2+), ReactJS et JQuery. Grâce à Onsen UI, des applications mobiles peuvent être créées à l’aide de technologies Web en tant de HTML5, CSS et JavaScript.
