1. Choix du développement

  • Tout d’abord, il faut choisir les technologies à utiliser pour la réalisation du projet, ils doivent comprendre en tant que le développement mobile, la base de données et le capteur d’alerte en cas d’un antivol du vélo.

    Développement mobile

    Onsen UI : Onsen UI est un Framework open source pour créer des applications Web progressives (PWAs) et des applications hybrides. Il est basé sur PhoneGap et Cordova. Les applications Web progressives (PWAs) sont créées avec le but d’offrir une fiabilité et une installation en atteignant n’importe qui, n’importe où et sur n’importe quel appareil avec une base de code unique. Les applications Web progressives installées s’exécutent dans une fenêtre distincte plutôt que dans un onglet du navigateur.

    Onsen UI peut être utilisé avec plusieurs frameworks en tant de JavaScript, AngularJS (1, 2+), ReactJS et JQuery. Grâce à Onsen UI, des applications mobiles peuvent être créées à l’aide de technologies Web en tant de HTML5, CSS et JavaScript. C’est pourquoi il a été décidé de réaliser une application web progressive, on peut l’utiliser soit par l’ordinateur ou bien pour le mobile en utilisant simplement ces technologies, au lieu d’utiliser des langages natifs pour le développement mobile et qu’ils ne permettent pas de créer des applications web. Notre cliente pourra faire d’usage de l’application n’importe quel appareil une fois installée.

    Gestionnaires de Bases de Données

    MySQL : C’est un système open source pour la gestion de base de données relationnelle (RDBMS). Une base de données relationnelle permet d’entreposer et d’organiser les données structurées sous forme de tableaux liés entre eux. Nous pouvons définir MySQL en tant qu’un Système de Gestion de Base de Données (SGBD), c’est-à-dire, un logiciel capable de manipuler les données d'une base de données, parmi les actions disponibles : créer, lire, mettre en jour et supprimer (CRUD en anglais).

    En raison des caractéristiques du projet à développer, il a été décidé d’utiliser une base de données relationnelle, parce que les données à gérer ainsi que doivent être organisées et que les tableaux seront liés les uns aux autres.
